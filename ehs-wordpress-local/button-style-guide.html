Usage Guidelines

Primary CTA Button: Use for main call-to-action buttons (Get Started, Request Quote, Contact Us in hero sections)
Secondary CTA Button: Use for secondary actions, footer CTAs, or when you need multiple buttons on the same page
Outline Button: Use for less prominent actions or when you want a lighter visual weight
Text Link Button: Use for "Read More", "Learn More" links or tertiary actions

Brand Colors Reference
