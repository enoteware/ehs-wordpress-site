# EHS Analytical Solutions - Branding Guide

**Date:** January 2025  
**Purpose:** Comprehensive branding guidelines for EHS Analytical Solutions and Construction Safety Consulting micro-site

---

## Main Site Branding (ehsanalytical.com)

### Color Palette

#### Primary Colors
- **Navy Blue (Primary):** `#1E405A` / `rgb(30, 64, 90)`
  - Use for: Headers, navigation, primary buttons, footer background, major headings
  - This is the main brand color

- **Gold/Yellow (Accent):** `#F6A90F` / `rgb(246, 169, 15)`
  - Use for: CTA buttons, highlights, important callouts, secondary buttons
  - Primary accent color for main site

#### Secondary Colors
- **White:** `#FFFFFF` / `rgb(255, 255, 255)`
  - Use for: Backgrounds, text on dark backgrounds

- **Light Gray (Background):** `#EAEAEA` / `rgb(234, 234, 234)`
  - Use for: Section backgrounds, alternating sections

- **Dark Gray (Text):** `#333333` / `rgb(51, 51, 51)`
  - Use for: Body text, secondary text

- **Dark Overlay:** `rgba(27, 37, 47, 0.76)`
  - Use for: Hero section overlays

### Typography

#### Font Families
- **Body Font:** `Roboto, sans-serif`
  - Size: 16px-18px
  - Line height: 1.6-1.8
  - Weight: Regular (400) for body, SemiBold (600) for emphasis

- **Heading Font:** `"Maven Pro", sans-serif`
  - H1: Bold (700), 42px-48px, White (on dark backgrounds) or Navy Blue
  - H2: Bold (700), 32px-36px, Navy Blue (#1E405A)
  - H3: SemiBold (600), 24px-28px, Navy Blue (#1E405A)

#### Font Loading
```html
<link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;600;700&family=Maven+Pro:wght@400;600;700&display=swap" rel="stylesheet">
```

### Logo

#### Main Logo
- **File:** `final-logo.svg`
- **Dimensions:** 422px × 166px (original)
- **Format:** SVG (scalable)

- Logo should link to homepage when in header

#### Logo Variations
1. **Full Logo** - Standard logo for header/navigation
2. **Logo with Division Text** - Logo + "A Division of EHS Analytical Solutions, Inc." below
3. **Favicon** - Simplified version or icon from logo for browser tab

### Button Styles

#### Primary Button
- **Background:** `#1E405A` (Navy Blue)
- **Text:** `#FFFFFF` (White)
- **Font:** Maven Pro SemiBold, 16px
- **Padding:** 14px 32px
- **Border Radius:** 4px
- **Hover:** Darken background by 10% (`#1A3448`)

#### Secondary Button (Accent)
- **Background:** `#F6A90F` (Gold/Yellow)
- **Text:** `#1E405A` (Navy Blue) or `#FFFFFF` (White)
- **Font:** Maven Pro SemiBold, 16px
- **Padding:** 14px 32px
- **Border Radius:** 4px
- **Hover:** Darken gold by 10% (`#DD9914`)

#### Button States
- **Default:** Solid background, white text
- **Hover:** Slightly darker background, subtle shadow
- **Active:** Pressed state (darker)
- **Disabled:** 50% opacity, no interaction

## Spacing and Layout

### Container Width
- **Maximum:** 1200px
- **Content Width:** 800px for text-heavy sections
- **Padding:** 60px-80px between major sections
- **Margin:** 20px sides on mobile, 40px on tablet+

### Grid System
- **Desktop:** 12-column grid
- **Tablet:** 2-column layouts
- **Mobile:** Single column, full width

### Section Spacing
- **Between Sections:** 80px vertical padding
- **Mobile:** 40px vertical padding
- **Within Sections:** 30px-40px spacing between elements

## Image Guidelines

### Image Sources
- **Primary:** Pexels API (professional stock photos)
- **Requirements:**
  - High quality (minimum 1920px width for hero images)
  - Professional, realistic (avoid overly staged stock photos)
  - Construction safety context
  - Diverse representation
  - Proper licensing (Pexels provides free commercial use)

### Image Categories

#### Homepage
- Construction site with safety professionals conducting inspection
- Workers in proper PPE
- Modern construction project

#### SSHO Services Page
- SSHO conducting safety inspection at military construction site
- Military installation in background
- Federal construction project

#### Safety Representatives Page
- Safety representative conducting jobsite inspection
- Safety meeting or toolbox talk
- Construction workers in safety gear

### Image Optimization
- **Format:** WebP (preferred), with JPEG fallback
- **Size:** Maximum 200KB per image
- **Lazy Loading:** Implement for images below the fold
- **Alt Text:** Descriptive, SEO-friendly, includes keywords when appropriate

### Image Dimensions
- **Hero Images:** 1920px × 600px (16:5 aspect ratio)
- **Section Images:** 1200px × 800px (3:2 aspect ratio)
- **Service Box Icons:** 64px × 64px or 80px × 80px
- **Certification Badges:** 150px × 150px

## Responsive Design

### Breakpoints
- **Mobile:** < 576px
- **Tablet:** 576px - 991px
- **Desktop:** 992px - 1199px
- **Large Desktop:** 1200px+

### Mobile Considerations
- Touch-friendly buttons (44x44px minimum)
- Readable text without zooming (16px minimum)
- Simplified navigation (hamburger menu)
- Stacked layouts
- Click-to-call phone numbers
- Fast loading (< 3 seconds)

---

## Accessibility

### WCAG 2.1 Level AA Compliance
- **Color Contrast:**
  - Text on background: Minimum 4.5:1 ratio
  - Large text (18pt+): Minimum 3:1 ratio
- **Keyboard Navigation:** All interactive elements accessible
- **Screen Readers:** Proper alt text, ARIA labels, heading hierarchy
- **Focus Indicators:** Visible focus states for keyboard navigation

## Brand Consistency Rules

1. **Always use main site's primary color** (`#1E405A`) for major elements
2. **Always use main site's accent color** (`#F6A90F`) for primary CTAs
3. **Maintain logo integrity** - don't modify colors or proportions
4. **Use consistent fonts** - Roboto for body, Maven Pro for headings
5. **Follow spacing guidelines** - maintain consistent padding and margins
6. **Ensure accessibility** - meet WCAG 2.1 AA standards
7. **Optimize for performance** - fast loading, optimized images
8. **Maintain relationship** - always show "A Division of EHS Analytical Solutions, Inc."
