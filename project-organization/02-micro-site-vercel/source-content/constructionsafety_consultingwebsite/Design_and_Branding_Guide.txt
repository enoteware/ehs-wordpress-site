TYPOGRAPHY
═══════════════════════════════════════════════════════════════

FONT FAMILIES:

Primary Font (Headings): Montserrat
- H1: Montserrat Bold, 42px-48px, Navy Blue
- H2: Montserrat Bold, 32px-36px, Navy Blue
- H3: Montserrat SemiBold, 24px-28px, Dark Gray
- Uppercase for major headings (optional)

Secondary Font (Body Text): Open Sans
- Body: Open Sans Regular, 16px-18px, Dark Gray
- Line height: 1.6-1.8 (good readability)
- Paragraph spacing: 1.2em

Alternative Font Pairing: 
- Headings: Roboto Condensed Bold
- Body: Lato Regular

FONT WEIGHTS:
- Bold (700): Major headings, emphasis
- SemiBold (600): Subheadings
- Regular (400): Body text
- Light (300): Captions (use sparingly)

MOBILE TYPOGRAPHY:
- Reduce heading sizes by 20-30% on mobile
- Maintain body text at 16px minimum (readability)
- Increase line height slightly on mobile (1.7-1.9)

LAYOUT & SPACING
═══════════════════════════════════════════════════════════════

PAGE LAYOUT:

Container Width: 1200px maximum

SECTION ALTERNATION:
- Alternate between white and light gray backgrounds
- Example: White section → Light gray section → White section

SERVICE BOXES / FEATURE CARDS
═══════════════════════════════════════════════════════════════

DESIGN:

Box Style:
- White background
- Light border (1px #E9ECEF) or subtle shadow
- Border radius: 8px
- Padding: 30px
- Icon/image at top
- Heading (H3)
- Description text (2-3 sentences)
- "Learn More" link or button at bottom

Icon Style:
- Size: 64x64px or 80x80px
- Color: Safety orange or navy blue
- Simple line icons or solid icons
- Centered above heading

Hover Effect:
- Subtle shadow increase
- Border color change to orange
- Slight scale increase (1.02x)
- Smooth transition (0.3s)

Layout:
- 3 columns on desktop (4 boxes = 1 row of 3, 1 row of 1 centered)
- 2 columns on tablet
- 1 column on mobile
- Equal height boxes

RESPONSIVE DESIGN REQUIREMENTS
═══════════════════════════════════════════════════════════════

BREAKPOINTS:

Mobile: < 576px
Tablet: 576px - 991px
Desktop: 992px - 1199px
Large Desktop: 1200px+

MOBILE CONSIDERATIONS:
✓ Touch-friendly buttons (44x44px minimum)
✓ Readable text without zooming (16px minimum)
✓ Easy-to-tap navigation
✓ Click-to-call phone numbers
✓ Simplified navigation menu
✓ Stacked layouts (no side-by-side on mobile)
✓ Fast loading (< 3 seconds)

TABLET CONSIDERATIONS:
✓ 2-column layouts where appropriate
✓ Readable text sizes
✓ Touch-friendly but more content density than mobile

DESKTOP CONSIDERATIONS:
✓ Take advantage of screen width
✓ 3-column layouts for service boxes
✓ Hover effects on interactive elements
✓ Larger, more impactful hero sections

ACCESSIBILITY REQUIREMENTS
═══════════════════════════════════════════════════════════════

WCAG 2.1 LEVEL AA COMPLIANCE:

Color Contrast:
✓ Text on background: Minimum 4.5:1 ratio
✓ Large text (18pt+): Minimum 3:1 ratio
✓ Test with contrast checker tools

Keyboard Navigation:
✓ All interactive elements accessible via keyboard
✓ Visible focus indicators
✓ Logical tab order

Screen Reader Support:
✓ Alt text for all images
✓ Proper heading hierarchy (H1 > H2 > H3)
✓ ARIA labels where needed
✓ Descriptive link text (no "click here")

Forms (if any):
✓ Proper label associations
✓ Error messaging
✓ Clear instructions
